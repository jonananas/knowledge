# Windows keyboard shortcuts

## Start feature/application
| Shortcut             | What it does              |
| -------------------- | ------------------------- |
| Win +                | Zoom in                   |
| Win -                | Zoom out                  |

## Word

| Shortcut             | What it does              |
| -------------------- | ------------------------- |
| Alt+Shift+Left/Right | Increase/Decrease Heading |
| Alt+Shift+Up/Down    | Move line up/down         |
